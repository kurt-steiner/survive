# 实操汇编

有这样一段 C 代码

```c
int add(int left, int right) {
    return left + right;
}

int main() {
    int temp1 = 125;
    int temp2 = 80;

    int sum = add(temp1, temp2);
    return sum;
}
```

1. `_main` 第一第二行: 创建新的栈空间
2. `_main` 第三行: 不要被 `sub` 迷惑，他的意思是 将 esp 往上移动，为创建局部变量 提供位置对齐，System V AMD64 ABI 约定，创建局部变量时，对栈空间进行16字节对齐
