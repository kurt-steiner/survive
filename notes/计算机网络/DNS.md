# DNS 服务

## 查询过程

DNS 的查询过程有两种

1. 递归 - 即 政府机关 帮你一步到位
2. 迭代 - 即 政府机关 相互推诿，甩锅推责，告诉你该去哪里查

现在我们一般 对 **本地域名服务器** 使用 递归查询，而 **本地域名服务器** 对 其他域名服务器 使用 迭代查询

## 解析过程

如果需要解析域名 a.b.c.d，
本地域名服务器 首先查看缓存中有没有，
如果没有，  
问 根域名服务器 (即中控台)，谁负责 d 域名的解析，根域名服务器给出 **服务器D**  
再问 **服务器D** ，c 域名由谁负责，给出 **服务器C**
再问 **服务器C** ，b 域名由谁负责，给出 **服务器B**
依次类推
